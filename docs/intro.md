# 欢迎使用 WikiTruth 文档

欢迎来到 WikiTruth 文档中心！

WikiTruth 是一个基于 **Oasis Sapphire** 隐私公链构建的去中心化证据市场，专注于机密证据的安全存储、可信交易与可控公开。

## 快速开始

- **[项目概览](/docs/overview)** - 了解 WikiTruth 的核心概念和特性
- **[技术文档](/docs/smart-contracts)** - 查看智能合约架构和技术实现
- **[功能细节](/docs/truth-box)** - 了解真相盒子、状态机制等功能
- **[常见问题](/docs/faq)** - 解答常见疑问

## 核心特性

- **去中心化**：核心业务逻辑由链上智能合约执行
- **匿名性**：保护举报人与证据上传者的身份隐私
- **安全性**：证据在本地加密，解密密钥由 Sapphire TEE 加密上链
- **经济激励**：通过代币激励和保密费机制驱动市场
- **社区治理**：DAO 治理、集体决策、全链透明
