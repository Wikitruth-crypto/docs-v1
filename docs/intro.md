# Welcome to WikiTruth Documentation

![Project](/docs/12-md.jpg)

WikiTruth is a decentralized evidence marketplace built on the **Oasis Sapphire** privacy public chain, focusing on the secure storage, trusted trading, and controlled disclosure of confidential evidence.

## Quick Start

- **[Overview](/docs/overview)** - Understand the core concepts and features of WikiTruth
- **[Technical Docs](/docs/smart-contracts)** - View smart contract architecture and technical implementation
- **[Features](/docs/truth-box)** - Learn about Truth Box, status mechanisms, and more
- **[FAQ](/docs/faq)** - Answers to common questions

## Core Features

- **Decentralization**: Core business logic is executed by on-chain smart contracts
- **Anonymity**: Protects the identity privacy of whistleblowers and evidence uploaders
- **Security**: Evidence is encrypted locally, and decryption keys are encrypted on-chain via Sapphire TEE
- **Economic Incentives**: Drives the market through token incentives and confidentiality fee mechanisms
- **Community Governance**: DAO governance, collective decision-making, fully transparent on-chain
